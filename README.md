# HR Analytics Dashboard (Power BI)

<br>**Overview**

This project is an interactive HR Analytics Dashboard built with **Microsoft Power BI**. The dashboard provides key HR metrics, employee demographics and attrition analysis to help  understand workforce trends and support data-driven decision making.<br><br>

### Dashboard features and key performance indicators<br>

| Feature | Description |
|----------|-------------|
| Total Employees | Displays the total number of employees in the organization. |
| Active Employees | Shows the number of currently active employees. |
| Attrition Count | Displays the total number of employees who have left the organization. |
| Attrition Rate | Shows the percentage of employees who have left the organization. |
| Average Age | Displays the average age of employees. |
| Average Experience | Shows the average years of employee work experience. |
| Department Analysis | Analyzes employee distribution and attrition across departments. |
| Salary Analysis | Compares attrition across different salary slabs. |
| Gender Analysis | Displays employee attrition by gender. |
| Job Role & Satisfaction | Examines attrition by job role and job satisfaction level. |
| Age Group Distribution | Visualizes the distribution of employees across different age groups. |
| Experience Trend | Shows how attrition varies with total years of experience. |
| Department-wise Employee Count | Displays the number of employees in each department. |
| Interactive Filtering | Allows users to filter the dashboard by age group and department for more detailed analysis. |

<br><br>**Dataset**

The dashboard uses an HR employee dataset (kaggle) containing information such as:

- Employee demographics
- Department
- Job role
- Salary slab
- Age
- Gender
- Total experience
- Job satisfaction
- Attrition status

<br>**Technologies used**

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling<br>

<br>**Data preparation**

The data preparation process included:
- Cleaning and transforming data using Power Query
- Removing duplicates and handling missing values
- Creating a calculated column and measure with DAX

<br>**Insights**

Some example insights that can be drawn from the dashboard:

- Operations has the largest workforce.
- Most employees are between **36 - 45 years old**.
- The highest attrition occurs in the **operations** department.
- Employees with lower salary slabs show higher attrition.
- Male employees represent the majority of attrition cases.
